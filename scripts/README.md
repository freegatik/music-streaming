# Скрипты генерации тестовых данных

## Генератор случайных тестовых данных

Скрипт `TestDataGenerator` автоматически создает случайные тестовые данные для приложения.

### Что генерируется:

- **20 артистов** - случайные имена из предопределенного списка известных артистов
- **40 альбомов** - по 2 альбома на каждого артиста
- **200 треков** - по 5 треков на каждый альбом
- **15 плейлистов** - случайные плейлисты от admin и testuser

### Данные:

- **Артисты**: имена из списка (Arctic Monkeys, The Beatles, Radiohead и т.д.), случайные страны, биографии с жанрами
- **Альбомы**: случайные названия, даты релиза (1990-2025), URL обложек
- **Треки**: случайные названия, длительность (120-420 секунд), жанры, URL аудио
- **Плейлисты**: случайные названия, описания, публичные/приватные

### Использование:

#### Вариант 1: Через аргумент командной строки

```bash
./gradlew bootRun --args="generate"
```

#### Вариант 2: Через скрипт

```bash
./scripts/generate_data.sh
```

### Примечание:

Генератор создает данные только если база данных пустая (нет артистов и треков). Для повторной генерации нужно очистить базу данных.
